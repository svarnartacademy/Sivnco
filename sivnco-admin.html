Setup Guide
One-time setup — takes about 10 minutes

Step 1 — Create Supabase tables

Go to your Supabase project → SQL Editor → paste and run this:

-- Gallery images table
CREATE TABLE IF NOT EXISTS sivnco_gallery (
  id TEXT PRIMARY KEY,
  url TEXT,
  alt TEXT,
  caption TEXT,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Enquiries table
CREATE TABLE IF NOT EXISTS sivnco_enquiries (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  name TEXT,
  email TEXT,
  service TEXT,
  message TEXT,
  read BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- Allow public read for gallery
ALTER TABLE sivnco_gallery ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Public can read gallery" ON sivnco_gallery FOR SELECT USING (true);
CREATE POLICY "Authenticated can write gallery" ON sivnco_gallery FOR ALL USING (true);

-- Allow public insert for enquiries (so your contact form can write)
ALTER TABLE sivnco_enquiries ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Public can insert enquiries" ON sivnco_enquiries FOR INSERT WITH CHECK (true);
CREATE POLICY "Authenticated can read enquiries" ON sivnco_enquiries FOR SELECT USING (true);
CREATE POLICY "Authenticated can update enquiries" ON sivnco_enquiries FOR UPDATE USING (true);

Step 2 — Create Storage bucket

Supabase → Storage → New bucket → Name: sivnco-images → Toggle Public ON → Create. That's it.

Step 3 — Connect this admin

In the sidebar: paste your Supabase Anon Key (Supabase → Settings → API → anon public key) and click Connect. Your URL is already filled in.

Step 4 — Form emails via Netlify

The contact form in sivnco-v5.html already has data-netlify="true" added. When you drag the folder to Netlify Drop, Netlify detects this automatically and sends you an email for every submission to your Netlify account email. No extra setup. Check your Netlify notification settings →

Step 5 — Deploy

Put sivnco-v5.html, sivnco-admin.html in the same folder. Drag to Netlify Drop. Done. Access your admin at yoursite.netlify.app/sivnco-admin.html
